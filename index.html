Para crear una ventana emergente (popup) en una aplicación de escritorio en Windows y Mac utilizando Python, puedes usar bibliotecas como Tkinter o PyQt. Aquí te muestro un ejemplo básico utilizando Tkinter, que es parte de la biblioteca estándar de Python y es bastante fácil de usar. Ejemplo con Tkinter Instala Tkinter (si no está instalado, en muchas distribuciones de Python ya viene incluido). Código básico para una ventana emergente: 




            import tkinter as tk
            from tkinter import messagebox
            
            def mostrar_popup():
                messagebox.showinfo("Título del Popup", "Este es un mensaje de popup.")
            
            # Crear la ventana principal
            ventana = tk.Tk()
            ventana.title("Aplicación Principal")
            ventana.geometry("300x200")
            
            # Botón para mostrar el popup
            boton_popup = tk.Button(ventana, text="Mostrar Popup", command=mostrar_popup)
            boton_popup.pack(pady=20)
            
            # Ejecutar la aplicación
            ventana.mainloop()
